# DataCo 供应链分析与决策支持

> 基于 180,519 个订单行的盈利、履约与库存分析｜Python + MySQL + Power BI

## 核心指标

- 57.31%｜订单级迟发率
- 12.03%｜整体利润率
- 18.71%｜亏损订单行占比
- 115.2 万｜基准安全库存资金

## 01 盈利｜深亏订单行集中贡献亏损

- **分析结果**：利润率低于 -50% 的订单行仅占 8.69%，却贡献 85.3% 的亏损；亏损总额 388 万，接近当前净利润 397 万。深亏占比在各品类近似均匀，表明问题主要与跨品类定价或履约规则相关。
- **管理建议**：对利润率低于 -100% 的订单行设置拦截规则，对 -50% 至 -100% 的订单行设置审批规则，并优先在 Fishing 与 Camping 品类试点。

## 02 履约｜承诺时效与实际能力存在偏差

- **分析结果**：First Class 承诺 1 天、实际恒为 2 天，导致订单级迟发率 100%；Second Class 实际时效与 Standard Class 接近。约18%的运输等级×市场×品类组合覆盖约80%的迟发订单行。
- **管理建议**：按历史履约能力校准承诺时效，测算显示订单级迟发率可由 57.3% 降至 31.6%；运输能力投入可集中于迟发量最大的 Standard Class 和仍有能力缺口的 Second Class。

## 03 库存｜安全库存资金集中于 B 类商品

- **分析结果**：在102个具有足够销售历史的可测算SKU中，A类7个SKU贡献77.6%的销售额。A 类仅占安全库存资金约 5%；B 类销售额占比为 17.9%，但安全库存资金占比约为 89%。
- **管理建议**：ABC-XYZ 差异化服务水平方案使测算资金增加 4.2%，因此优先方向应为压缩 B 类补货提前期。提前期减半情景可释放约 30 万资金，BZ 组合可进一步评估按单采购策略。
