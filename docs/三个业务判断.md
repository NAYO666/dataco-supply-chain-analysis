# 业务分析结论

本文档汇总盈利、履约和库存三个专题的分析问题、量化结果、建议动作与数据边界。相关计算分别记录在三份正式 Notebook 中。

## 盈利：亏损集中于少量深亏订单行

### 分析问题

亏损是否集中于特定品类、区域或 SKU，以及哪些订单行对整体亏损贡献最大。

### 分析结果

- 亏损订单行占全部订单行的 18.71%。
- 品类×区域层面仅 60 个组合净亏，合计净亏 1.45 万，占全部亏损 0.4%。
- 利润率低于 -50% 的订单行占 8.69%，贡献 85.3% 的亏损。
- 利润率低于 -100% 的订单行贡献 53% 的亏损。

### 建议动作

设置订单行利润率下限，对利润率低于 -100% 的交易进行拦截，对 -100% 至 -50% 的交易触发审批。根据深亏金额，可优先在 Fishing 和 Camping 品类验证规则效果。

### 数据边界

数据集不包含运费成本，因此负利润用于识别亏损订单行，不用于判断亏损是否由运费造成。

## 履约：承诺时效与实际配送能力存在系统性偏差

### 分析问题

迟发是否集中于特定运输等级和市场，以及调整承诺时效能够减少多少迟发。

### 分析结果

- 有效发货订单的订单级迟发率为 57.31%。
- First Class 承诺 1 天、实际配送 2 天，订单级迟发率为 100%。
- Second Class 承诺 2 天、实际平均 4.00 天，与 Standard Class 的 3.99 天接近。
- 约18%的运输等级×市场×品类组合覆盖约80%的迟发订单行。
- 迟发订单行对应 GMV 约 2,013 万，占总 GMV 的 57.2%。

### 建议动作

首先根据历史履约能力校准运输等级的承诺时效，再将运营改善资源投入 Second Class 和 Standard Class 的剩余能力缺口。历史反事实测算显示，按各运输等级实际均值校准承诺后，订单级迟发率由 57.3% 降至 31.6%。

### 数据边界

数据集不包含罚金和客户流失字段，因此受影响 GMV 表示迟发覆盖的交易规模，不等同于直接经济损失。情景测算保持历史实际配送天数不变，用于评估承诺口径变化，不构成因果推断。

## 库存：安全库存资金主要集中于 B 类商品

### 分析问题

统一服务水平与 ABC-XYZ 分层策略的安全库存资金差异，以及资金占用的主要驱动因素。

### 分析结果

- 在102个具有足够销售历史的可测算SKU中，A类7个SKU贡献77.6%的销售额。A 类仅占安全库存资金约 5%。
- B 类销售额占比为 17.9%，安全库存资金占比约为 89%。
- 统一 95% 服务水平的安全库存资金为 115.2 万。
- ABC-XYZ 分层服务水平策略的安全库存资金为 120.1 万，较基准增加 4.2%。
- 仅降低 C 类服务水平时，安全库存资金为 112.6 万，较基准减少 2.2%。

### 建议动作

优先降低 B 类商品的补货提前期。测算显示，提前期减半可使安全库存资金降低约 29%，对应约 30 万；BZ 类可进一步评估按单采购策略。

### 数据边界

数据集不包含库存快照和供应商提前期。安全库存使用 `SS = z × σ × √L` 进行策略测算，商品单价作为库存资金成本近似，补货提前期统一设为 2 周。该结果用于比较策略，不代表现有库存审计结果。
