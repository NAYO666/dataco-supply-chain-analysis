# 字段筛选与数据治理

原始数据包含 180,519 个订单行和 53 个字段。数据粒度为订单行：每行对应一个订单中的一个商品项；`Order Id` 去重后共有 65,752 个订单，平均每个订单包含 2.75 个订单行。

数据清洗保留 31 个分析字段，删除 22 个隐私、低质量、冗余或与分析目标无关的字段。

## 保留字段

### 键与日期

| 原始字段 | 标准名称 | 用途 |
|---|---|---|
| Order Item Id | 订单行 ID | 事实表主键 |
| Order Id | 订单 ID | 订单级履约指标去重键 |
| Customer Id | 客户 ID | 客户维度外键 |
| Product Card Id | 商品 ID | 商品维度外键 |
| order date (DateOrders) | 下单日期 | 日期维度来源 |
| shipping date (DateOrders) | 送达日期 | 履约周期与到货日期 |

## 删除字段

### 隐私、打码或无有效信息字段（8个）

- `Customer Email`：打码值。
- `Customer Password`：打码值。
- `Customer Fname`、`Customer Lname`：个人姓名字段。
- `Customer Street`：地址字段。
- `Product Description`：全空字段。
- `Product Image`：外部图片 URL。
- `Order Zipcode`：大量缺失且未出现在字段说明文件中。

### 客户注册地冗余字段（6个）

`Customer City`、`Customer Country`、`Customer State`、`Latitude`、`Longitude`、`Customer Zipcode`。履约分析使用订单目的地字段，因此不保留客户注册地。

### 完全重复字段（5个）

| 删除字段 | 保留字段 |
|---|---|
| Benefit per order | Order Profit Per Order |
| Order Item Product Price | Product Price |
| Product Category Id | Category Id |
| Order Customer Id | Customer Id |
| Order Item Cardprod Id | Product Card Id |

以上字段通过逐行相等检查确认完全重复。

### 常量或非分析范围字段（3个）

- `Product Status`：所有订单行均为 0。
- `Type`：支付方式，不参与三个分析专题。
- `Sales per customer`：客户累计销售额，可由订单行重新聚合。

## 数据治理结果

- 原始字段：53 个。
- 保留字段：31 个。
- 删除字段：22 个。
- 原始订单行：180,519 个。
- 清洗后订单行：180,519 个。
- `order_item_id`：无重复。
- 关键分析字段：无空值。
