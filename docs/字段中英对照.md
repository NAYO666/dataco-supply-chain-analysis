## 命名决策说明

**① `shipping_date` = 送达日期**
数据验证显示，`shipping_date − order_date` 恒等于 `days_for_shipping_real`，因此该字段表示配送完成日期。

**② `shipping_mode` = 运输等级**
取值为 Standard / First / Second Class 和 Same Day，表示按承诺时效划分的服务等级。

**③ `department` = 商品大类**
取值包括 Fan Shop、Apparel、Golf 和 Footwear，表示商品分类层级。

**④ 地理字段使用“订单”前缀**
地理字段描述订单记录中的目的地。使用“订单城市”“订单国家”和“订单州省”可以同时覆盖完成、取消和疑似欺诈订单。

**⑤ `order_item_profit_ratio` = 行级利润率**
Power BI 模型包含同名度量值 `[利润率]`（总利润 ÷ 净收入）。为避免 DAX 引用混淆，数据列命名为“行级利润率”：
- **数据列**：用于订单行亏损深度分类。
- **度量值**：用于计算汇总利润率。
